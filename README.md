# formatDate

Библиотека для форматирования даты и времени под определенный стандарт

## formatDateToRu()

формат RU даты ДД/ММ/ГГ ЧЧ:ММ

## formatDateToUs()

формат US даты ММ-ДД-ГГ ЧЧ:ММ
